# Space Invaders (from planet Zig)
This is a recreation of the classical game Space Invaders, made as a simple exercise to learn Zig
Currently only working on windows 
To run just clone the repo and execute 
```
zig build run
```
# Controls
- H to go left
- L to go right
- Space bar to shoot
